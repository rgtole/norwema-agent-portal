# Marathi Culture & Blogger Skill

This skill allows the Agent to:
1. Generate cultural history and story drafts based on North West Marathi community prompts.
2. Formulate Marathi Devnagari translations of key headers or summaries (e.g. using standard phrases like "गणेशोत्सव २०२६", "हार्दिक स्वागत").
3. Format output as beautiful HTML/Markdown cards.
4. Insert completed blog posts into the `blogs` collection via the MCP Server.

## Style Guidelines
- Maintain a warm, community-focused, and respectful tone.
- Incorporate both English and Marathi translations for important titles or slogans.
